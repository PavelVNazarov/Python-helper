Активация виртуального окружения в Windows
Создание виртуального окружения (если еще не создано):
В командной строке, находясь в директории проекта, выполни:
python -m venv venv

Это создаст папку venv с виртуальным окружением.
Активация виртуального окружения:
Выполни следующую команду:

venv\Scripts\activate

Добавление зависимостей:

Если у тебя уже установлены зависимости в виртуальном окружении, 
ты можешь автоматически создать файл requirements.txt с помощью команды:
pip freeze > requirements.txt


телеграмм бот и джанго
pip install python-telegram-bot

В Вашем Django проекте создайте новое приложение (если у Вас его еще нет):
python manage.py startapp telegram_bot



Чтобы создать суперпользователя в Django, выполните следующие шаги:
Откройте терминал и перейдите в директорию Вашего проекта Django.
Запустите следующую команду:

python manage.py createsuperuser

Вам будет предложено ввести имя пользователя, адрес электронной почты и пароль для суперпользователя. Убедитесь, что пароль соответствует требованиям безопасности.
После успешного создания суперпользователя Вы сможете войти в административную панель Django по адресу http://localhost:8000/admin/ с использованием учетных данных, которые Вы только что создали.
